# Kopach3D

Сборка и запуск движка на Marching Cubes.

## Зависимости

- CMake >= 3.20
- C++20
- OpenGL 4.6
- Git

Библиотеки скачиваются автоматически: GLFW, GLM, ImGui.

## Сборка

mkdir build
cd build
cmake ..
cmake --build .

## Запуск

build\bin\Debug\Kopach3D.exe

## Управление

WASD - движение
Q/E - вверх/вниз
Мышь - поворот
TAB - курсор
F1 - wireframe
ESC - выход
